# Hotel Management System

## Overview

The Hotel Management System is a responsive web application developed to provide users with an easy and interactive hotel browsing and room booking experience.

The website allows users to explore available rooms, view hotel information, and submit booking requests through a user-friendly interface.

## Features

- Responsive Home Page
- Room Listings and Details
- Hotel Information Section
- Room Booking Form
- Interactive User Interface
- Mobile-Friendly Design

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

- index.html – Homepage
- rooms.html – Room details and listings
- booking.html – Room booking page
- contact.html - Contact page
- style.css – Website styling
- script.js – Interactive functionality
- images/ – Hotel and room images
